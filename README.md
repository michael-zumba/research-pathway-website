# Research Pathway Website

This is the source code for the Research Pathway Limited website.

## Domain
This site is configured to run on `www.researchpathway.co.nz`.

## Deployment
This site is ready for GitHub Pages or Netlify.
- **CNAME**: Included for custom domain support.
- **Structure**: Static HTML/CSS.

## Directory Structure
- `/`: Main English pages.
- `/cn/`: Chinese translated pages.
- `/google_sites_resources/`: Tools and guides for the legacy Google Sites implementation.
